# 权限列表

> Wayne 提供了完整的权限管理体系，也提供了快捷的权限导入方案，
因此在此列出 Wayne 的所有权限和意义，管理员用户可以在 后台->权限->权限列表 创建相关权限，在 后台->权限->角色列表 编辑相关角色绑定相关权限。

Wayne 资源权限主要分为三类，基础权限，Wayne 自定义资源权限以及 Kubernetes 资源权限
